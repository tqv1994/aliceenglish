== Description ==

Have an upcoming event or an interesting sale youâ€™d like to share on your WordPress website? The Themify Event Post Type plugin allows users to create an event specific post type. Here's what it can do:

- Event Details Area: you can input the day, start & end time, and location
- Map Input Field: you can input an address to display a Google Map
- Buy Button Link: you can enter a link to a ticket buying page
- Repeat Option: you can opt to automatically replicate the event weekly or even daily
- When used with the <a href="https://themify.me/builder">Themify Builder</a> plugin it'll also come with its own Event Post module.


== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!
